# Chat to Eva - Enhanced Voice Assistant

## Overview

Chat to Eva is an advanced voice assistant application that integrates speech-to-text and text-to-speech functionalities with OpenAI's API. It features a Live2D model for visual interaction and supports various tools for enhanced user experience.

## Features

- **Speech-to-Text and Text-to-Speech**: Convert spoken words to text and vice versa using OpenAI's API.
- **Live2D Model Integration**: Visual representation of the assistant with dynamic expressions.
- **Clipboard and Webcam Integration**: Supports clipboard text/image processing and webcam image capture.
- **Tool Management**: Includes tools for managing todo lists, memory cache, and web navigation.
- **Workflow Execution**: Run workflows or generate code based on user prompts.

## Usage

- **Interacting with Eva**: Type or speak your queries into the input field. Use the buttons to send messages or start recording.
- **Tool Settings**: Configure API keys and endpoints in the collapsible tool settings section.
- **Clipboard and Webcam**: Use the clipboard button to paste content or enable the webcam for image processing.
